# Primo-sorted-num-with-unit-test

### Dev Dependencies: 
you must have
  * @types/jest: 30.0.0 or more
  * jest:        30.2.0 or more
  * ts-jest:     29.4.6 or more
  * ts-node:     10.9.2 or more
  * typescript:  5.9.3  or more

# Prerequisites
  ### Node.js and npm
  please make sure you're installed Node.js. if you don't, you can follow the step below:
  1. Open Node.js Website 
  _https://nodejs.org/en/download_

  2. download the package in the way you like (would recommend by windown installer if you were use windown pc or laptop) 
  <img width="957" height="331" alt="image" src="https://github.com/user-attachments/assets/132bb6cc-69de-4c0b-813d-97bff88fc14b" />

  3. Open the installation, config and install as you want.

  ## To check if your node.js was installed succesfully or not on your OS. 
  you can check by the step below: 
  
  1. Open your command prompt (administator is optional)

  2. type the command below into the command prompt and run
  ```sh
    node --version
  ```
  ```sh
    npm --version
  ```
  if your command showing something like this. congratulation, you have node.js and npm ready in your OS now!
  <img width="1202" height="128" alt="image" src="https://github.com/user-attachments/assets/7bb60c04-1afb-45cc-9413-41a41213e066" />

  Question (in case you were curious); Why **npm** were in my computer, I didn't installed yet!?
  * well, you actually did. by installing **node.js** into your OS; it were budled by default.

  ### TypeScript
  to downlode it, your OS need to recognize the npm command first. if your OS already recognize the command, please follow the step below:

  1. Open the command prompt (administator is optional)
  2. type the command below into the command prompt and run
  ```sh
    npm install -g typescript
  ```
  then wait untill it show like the picture below
  <img width="1200" height="87" alt="image" src="https://github.com/user-attachments/assets/b836cc8d-5dca-40ae-915e-a8beb8378161" />
